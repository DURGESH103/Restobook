## Responsive Behavior

### Desktop (> 1024px)
- Full navbar with all links
- 4-column footer
- 3-column menu grid
- Side-by-side form fields

### Tablet (768px - 1024px)
- Condensed navbar
- 2-column footer
- 2-column menu grid
- Side-by-side form fields

### Mobile (< 768px)
- Hamburger menu
- 1-column footer
- 1-column menu grid
- Stacked form fields
- Full-width buttons
